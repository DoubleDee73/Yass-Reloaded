# Yass Reloaded 2026.4 Release Notes

These release notes summarize the **user-facing changes** between `2026.3_release` and `2026.4_release`.

## Highlights

- New **transcription and alignment workflows** with **OpenAI Whisper** and **WhisperX**
- New local stem separation via **audio-separator** as an alternative to **MVSEP**
- Major improvements to the **Absolute Pitch Editor**
- Strongly reworked **note-tapping recording mode**
- New **GitHub Pages documentation** with structured help pages

## Editor and Absolute Pitch View

- New **Align to Melody** command
  - aligns selected notes to the detected pitch line
  - works directly with the pitch detection already available in the editor
  - can align in the detected octave in recording and wizard contexts
- Improved navigation in the absolute grid
  - horizontal timeline navigation through the track/overview area
  - new vertical pitch navigation with zoom and viewport control
- Improved editing
  - smoother note movement
  - better multi-selection
  - additive and drag-based selection improvements
- Off-screen indicators for notes outside the visible pitch range
  - notes can be indicated with arrows above or below the visible grid
- Additional editor interaction improvements
  - hand-tool style panning in the sheet
  - better mouse-over zones for move and resize
  - visual improvements for navigation and scroll controls
- New dialogs:
  - **Move Cursor**
  - **Move Remainder**
- Improved Insert Note workflow
  - notes can now be inserted with text directly
  - multi-syllable text uses the existing lyric splitting logic

## Note-Tapping Recording
- The tapping workflow was significantly reworked:
  - rolling recording viewport instead of only page-based navigation
  - tap queue with upcoming syllables/notes
- Recording can now start more intelligently:
  - from the selected note
  - or from the current cursor position if no note is selected
- Improved interrupt and continue flow
  - pressing `ESC` can now lead to keeping, discarding, or continuing the current tapping session
- Better tap post-processing
  - Align to Melody will be applied directly to tapped notes
  - untapped notes can be preserved or repositioned depending on context

## Transcription and Lyrics Alignment

- New transcription support in the editor:
  - **Align Notes with Transcription** via **OpenAI**
  - **Align Notes with Transcription** via **WhisperX**
- Support for cached transcription files
  - existing results can be reused instead of calling the service again
- WhisperX can now also **rebuild notes directly from transcript timestamps**
- Better handling of lyrics from clipboard and wizard flows
  - diff/compare dialog for wizard lyrics
  - corrected wizard lyrics are applied more reliably before note rebuild

## Vocal Separation and External Tools

- New local stem separation with **audio-separator**
  - local alternative to MVSEP
  - multiple model types can be selected
  - dedicated configuration page under **External Tools**
  - health check support
  - package update button inside Yass Reloaded
- **MVSEP** was extended
  - additional model and algorithm options
  - better status and resume handling
  - improved dialog for starting and monitoring jobs
- **WhisperX** was expanded significantly
  - dedicated configuration area
  - health check
  - in-app package update
  - auto-configuration for device, compute type and model based on detected GPU/VRAM
- **OpenAI transcription** was integrated
  - API key and model selection in settings
  - more robust API error handling
- External tool configuration overall has been improved
  - more consolidated settings pages
  - clearer status displays
  - additional built-in help and documentation

## Create Song Wizard

- The wizard can now automate a much larger part of the pipeline:
  - download
  - separation
  - transcription
  - lyric transfer
- New or expanded **Separate Vocals + Transcribe** flow
  - works with **MVSEP** or **audio-separator**
  - then transcribes with **WhisperX**
- Improved YouTube and download flow
  - better reuse of previously downloaded files
  - more reliable FFmpeg and yt-dlp handling
  - better temporary file and reuse logic based on the YouTube ID
- Improved wizard lyric transfer
  - lyric edits in the wizard are applied more cleanly before note generation
- Wizard step visibility and availability now react more cleanly to the configured tool setup
  - for example, separation and transcription only appear when the required tools are configured correctly

## Song Header, Playback and General Editor UX

- Song Header and lyrics area were reorganized
- START and END fields improved
- `YassSheetInfo` and editor overlays were reworked in multiple places
  - better visibility of important information
  - fewer distracting overlays during normal playback

## Cover, Metadata and Library

- New Library/Extras action:
  - **Search fanart.tv Cover**
- fanart.tv integration:
  - searches matching covers via MusicBrainz/fanart.tv
  - preview dialog for results
  - download into the song folder
  - automatic update of the `#COVER` tag
  - handling of existing cover files via Replace / Overwrite / Cancel
- New **fanart.tv API key** setting

## Documentation

- New structured project documentation via **GitHub Pages**
- Expanded documentation now includes:
  - Installation
  - Quick Start
  - Song Editor
  - Note Editing
  - Recording / Tapping
  - Settings
  - Wizard
  - External Tools
  - Keyboard Shortcuts
  - FAQ / Troubleshooting

## Bug Fixes and Stability

- Many fixes in the recording flow
- Multiple fixes in the wizard, audio conversion, and tool integrations
- Various UI and layout fixes in SongHeader, Sheet, Lyrics, and the video dialog
