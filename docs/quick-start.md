# Quick Start

This page is meant for users who want to get productive quickly.

## 1. Open or Create a Song

You can either:

- open an existing UltraStar song from the library
- or create a new song with the creation wizard

If you use the wizard, it can also help with:

- downloading or locating source media
- stem separation
- transcription
- opening the result directly in the editor

For existing collections, you can also start from the library and jump into the editor only when a song needs attention.

## 2. Confirm the Song Basics

Make sure the song has the practical basics it needs:

- a usable main audio file
- sensible metadata
- lyrics or a plan for how to create them
- a workflow for timing the notes

Track assignments like `#VOCALS` and `#INSTRUMENTAL` can be very helpful later, but they are not the first thing most users need to worry about.

## 3. Edit Notes and Lyrics

Typical basic tasks:

- select notes in the sheet
- move notes in time or pitch
- resize notes
- edit lyrics in the lyrics table
- adjust page breaks

If you prefer octave-stable editing and scrolling, try the absolute pitch view.

## 4. Use Playback

Playback is essential for timing and alignment work.

Common usage:

- play current selection
- play current page
- move the cursor
- check the visible pitch lines against existing notes

Common shortcuts:

- `Space` for selection playback
- `P` for page playback
- `Esc` to stop
- `Ctrl-R` to start recording/tapping

## 5. Record or Tap Notes

For note timing workflows, use the note-tapping recording mode.

Typical flow:

1. select the note or range
2. start note-tapping recording
3. tap along with the music
4. confirm the result
5. refine with manual editing or Align to Melody

See also:

- [Recording and Tapping](recording/tapping.md)
- [Keyboard Shortcuts](editor/keyboard-shortcuts.md)
- [Library Overview](library/overview.md)

## 6. Save

Save your changes after editing and verify:

- song text file
- cover / media references
- page breaks
- note alignment
