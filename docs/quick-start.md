# Quick Start

This page is meant for users who want to get productive quickly.

## 1. Open or Create a Song

You can either:

- open an existing UltraStar song from the library
- or create a new song with the creation wizard

## 2. Check Audio Assignment

Make sure the relevant files are assigned correctly:

- `#AUDIO`
- `#VOCALS`
- `#INSTRUMENTAL`
- optional video or cover assets

## 3. Edit Notes and Lyrics

Typical basic tasks:

- select notes in the sheet
- move notes in time or pitch
- resize notes
- edit lyrics in the lyrics table
- adjust page breaks

## 4. Use Playback

Playback is essential for timing and alignment work.

Common usage:

- play current selection
- play current page
- move the cursor
- check the visible pitch lines against existing notes

## 5. Record or Tap Notes

For note timing workflows, use recording / tapping mode.

Typical flow:

1. select the note or range
2. start recording
3. tap along with the music
4. confirm the result
5. refine with manual editing or Align to Melody

See also:

- [Recording and Tapping](recording/tapping.md)

## 6. Save

Save your changes after editing and verify:

- song text file
- cover / media references
- page breaks
- note alignment
