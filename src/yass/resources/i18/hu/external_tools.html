Külső eszközök

A Yass Reloaded több opcionális külső eszközzel is együttműködik a vokálszétválasztás és dalszöveg-transzkripció érdekében. Nyisd meg a Beállítások > Külső eszközök menüt a konfiguráláshoz.

FFmpeg

Az FFmpeg szükséges az audiokonverzióhoz (AAC, OGG, OPUS stb.) és a vokálszétválasztási folyamathoz. A Yass automatikusan is letöltheti az FFmpeg-et, vagy megadhatsz egy meglévő telepítést itt: Külső eszközök > Helyek.

  • Letöltés: https://ffmpeg.org/download.html
  • Windows alatt a ffmpeg.exe-t tartalmazó bin mappát add meg.

Python futtatókörnyezetek

Több helyi eszköz Pythonra épül. A Külső eszközök > Helyek alatt megadhatsz egy alapértelmezett Python executable értéket. A Yass ezt közös fallbackként használja a kisebb Python-segédekhez, valamint alap interpreterként a tool-specifikus környezetek létrehozásához.

  • Ha egy eszköz Python-mezője üres, a Yass a Helyek alatt beállított alapértelmezett Pythonra vált vissza.
  • Ha a WhisperX vagy audio-separator esetén az Update package gombra kattintasz üres tool-specifikus Python mező mellett, a Yass automatikusan létrehoz egy dedikált virtuális környezetet, és elmenti annak Python útvonalát.
  • Így a nagy függőségek, például a WhisperX és az audio-separator elkülönülnek, miközben a kisebb Python-segédek továbbra is használhatják a közös alapértelmezett Pythont.

yt-dlp

A yt-dlp-t a dal-létrehozó varázsló használja hang letöltésére YouTube-ról és más platformokról. Nincs szükség API-kulcsra.

  • Telepítés: pip install yt-dlp vagy önálló bináris letöltése.
  • A kodeket, felbontást és hangformátumot a Beállítások > Varázsló > yt-dlp alatt állíthatod.

MVSEP (online vokálszétválasztás)

Az MVSEP egy felhőalapú vokálszétválasztó szolgáltatás. Feltölti az audiót az MVSEP API-ba, szétválasztja stemekre, majd letölti az eredményt. Fiók és API-token szükséges.

  1. Regisztrálj a https://mvsep.com oldalon, és másold ki az API-tokenedet a profilodból.
  2. Nyisd meg a Beállítások > Külső eszközök > MVSEP panelt, és illeszd be a tokent.
  3. Válassz szétválasztási modellt. A Vokálmodell általában maradhat az alapértéken, az Audioformátum pedig a visszakapott stemek formátumát határozza meg.
  4. A varázslóban kattints a Separate Vocals + Transcribe gombra a teljes folyamat futtatásához.
  5. A szerkesztőben már megnyitott dalokhoz használd a Szerkesztés > Hang szétválasztása menüt.

audio-separator (helyi vokálszétválasztás)

Az audio-separator előre betanított szétválasztó modelleket futtat helyben a gépeden, internet vagy API-kulcs nélkül. Ugyanazokat az MDX-Net, VR Architecture és Demucs modelleket támogatja, mint az Ultimate Vocal Remover (UVR).

  1. Telepíts Python 3.9-et vagy újabbat.
  2. Telepítsd az audio-separatort:
    • Csak CPU: pip install audio-separator[cpu]
    • NVIDIA GPU: pip install audio-separator[gpu]
  3. Nyisd meg a Beállítások > Külső eszközök > audio-separator panelt.
  4. Add meg az eszköz Python executable értékét, vagy hagyd üresen, hogy a Helyek alatt megadott alapértelmezett Python legyen használva.
  5. Ha üres tool-specifikus Python mezővel használod az Update package gombot, a Yass automatikusan létrehoz egy dedikált audio-separator virtuális környezetet, és kitölti a mezőt.
  6. Válassz modellt. Az ismert modellek olvasható néven jelennek meg, az ismeretlenek pedig fájlnévként maradnak. Ajánlott modellek:
    • Kim_Vocal_2.onnx - gyors, jó minőség (alapértelmezett)
    • MDX23C-8KFFT-InstVoc_HQ.ckpt - magas minőség
    • UVR-MDX-NET-Inst_HQ_3.onnx - alternatíva
    • BS-Roformer-Viperx-1297.ckpt - legjobb minőség, lassabb
  7. Opcionálisan megadhatsz modell-gyorsítótár mappát. Ide kerülnek a letöltött modellek, így nem kell őket újra letölteni.
  8. Az Output format azt a nyers formátumot szabályozza, amelyet az audio-separator először kiír, mielőtt a Yass a stemeket a beállított célformátumba konvertálja.
  9. Kattints a Test configuration gombra a beállítás ellenőrzéséhez.
  10. Sikeres ellenőrzés után a Separate Vocals + Transcribe gomb elérhetővé válik a varázslóban. Ha az MVSEP és az audio-separator is be van állítva, az audio-separator élvez elsőbbséget.

OpenAI Whisper (felhős transzkripció)

Az OpenAI Whisper API időbélyeges szavakkal transzkribálja az audiót, majd az eredményt a szerkesztő hangjegyeire alkalmazza. OpenAI API-kulcs szükséges.

  1. Szerezz API-kulcsot itt: https://platform.openai.com/api-keys.
  2. Nyisd meg a Beállítások > Külső eszközök > OpenAI panelt, és add meg a kulcsot.
  3. Válassz transzkripciós modellt (pl. whisper-1) és opcionálisan nyelvkódot.
  4. A szerkesztőben használd a Szerkesztés > Align Notes (Transcription) menüt az aktuális dal transzkripciójához és igazításához.

WhisperX (helyi transzkripció)

A WhisperX helyben futtat Whisper-alapú transzkripciót szó szintű időbélyegekkel. Nem igényel internetet vagy API-kulcsot.

  1. Telepíts Python 3.9-et vagy újabbat.
  2. Telepítsd a WhisperX-et: pip install whisperx
  3. Nyisd meg a Beállítások > Külső eszközök > WhisperX panelt.
  4. Add meg a WhisperX Python executable értékét, vagy hagyd üresen, hogy a Helyek alatt megadott alapértelmezett Python legyen használva.
  5. Ha üres WhisperX Python mezővel használod az Update package gombot, a Yass automatikusan létrehoz egy dedikált WhisperX virtuális környezetet, és elmenti annak runtime-ját.
  6. Válaszd ki a hívási módot (python -m whisperx ajánlott).
  7. Válassz modellt (pl. large-v2) és eszközt (cpu vagy cuda).
  8. Kattints a Test configuration gombra az ellenőrzés futtatásához. Az eredmény mentésre kerül, és engedélyezi a Separate Vocals + Transcribe gombot a varázslóban.
  9. A szerkesztőben használd a Szerkesztés > Align Notes (Transcription) menüt, majd válaszd a WhisperX lehetőséget.

Dal-létrehozó varázsló: teljes folyamat

A varázsló a teljes folyamatot automatikusan végig tudja futtatni:

  1. Adj meg egy YouTube URL-t a YouTube oldalon, majd töltsd le az audiót.
  2. A Dalszöveg oldalon kattints a Separate Vocals + Transcribe gombra.
  3. A Yass WAV-vá alakítja az audiót, szétválasztja a vokálokat (audio-separator vagy MVSEP), majd transzkribálja őket (WhisperX).
  4. A dalszöveg automatikusan kitöltődik. Folytatás előtt még javíthatod.
  5. Fejezd be a varázslót, és a dal előre kitöltött időbélyegekkel jön létre a transzkripció alapján.

Követelmények: audio-separator (helyi, sikeres ellenőrzés) vagy MVSEP (beállított API-token), valamint WhisperX (sikeres ellenőrzés).
