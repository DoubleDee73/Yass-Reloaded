Créer une nouvelle chanson

Un fichier audio (par exemple MP3) de la chanson est maintenant nécessaire. Veuillez noter que si le fichier a été encodé avec un débit binaire variable (VBR), les anciennes versions d’UltraStar pourraient rencontrer des problèmes. Si la source originale provient de YouTube, il est recommandé de choisir Opus ou Ogg, car les fichiers M4A ou MP3 peuvent présenter de légères différences dans le "gap" en raison du transcodage.

Veuillez choisir le fichier MP3.
