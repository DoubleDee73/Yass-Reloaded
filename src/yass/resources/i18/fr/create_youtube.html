URL YouTube

Si vous créez une chanson à partir d'un fichier YouTube, vous pouvez saisir l'adresse YouTube ci-dessous. L'ID YouTube est également suffisant (par ex. https://www.youtube.com/watch?v=u_sIfs7Yom4

Yass Reloaded téléchargera les fichiers audio et vidéo dans la qualité et le format que vous avez spécifiés dans vos préférences

Les sous-titres seront également téléchargés et utilisés comme paroles, sauf s'ils ont été générés automatiquement
