Outils externes

Yass Reloaded s'intègre à plusieurs outils externes optionnels pour la séparation vocale et la transcription des paroles. Ouvrez Paramètres > Outils externes pour les configurer.

FFmpeg

FFmpeg est nécessaire pour la conversion audio (AAC, OGG, OPUS, etc.) et pour le flux de séparation vocale. Yass peut télécharger FFmpeg automatiquement, ou vous pouvez pointer vers une installation existante dans Outils externes > Emplacements.

  • Téléchargement : https://ffmpeg.org/download.html
  • Sous Windows, indiquez le dossier bin contenant ffmpeg.exe.

Runtimes Python

Plusieurs outils locaux utilisent Python. Dans Outils externes > Emplacements, vous pouvez définir un exécutable Python par défaut. Yass l'utilise comme solution commune pour les petits assistants Python et comme interpréteur de base pour créer des environnements spécifiques par outil.

  • Si le champ Python d'un outil est vide, Yass utilise le Python par défaut défini dans Emplacements.
  • Si vous cliquez sur Update package pour WhisperX ou audio-separator sans runtime Python spécifique défini, Yass crée automatiquement un environnement virtuel dédié pour cet outil et y enregistre le chemin Python.
  • Ainsi, les dépendances lourdes comme WhisperX et audio-separator restent isolées, tandis que les petits outils Python peuvent partager le Python par défaut.

yt-dlp

yt-dlp est utilisé par l'assistant de création de chanson pour télécharger l'audio depuis YouTube et d'autres plateformes. Aucune clé API n'est nécessaire.

  • Installation : pip install yt-dlp ou téléchargement du binaire autonome.
  • Configurez codec, résolution et format audio dans Paramètres > Assistant > yt-dlp.

MVSEP (séparation vocale en ligne)

MVSEP est un service cloud de séparation vocale. Il envoie votre audio à l'API MVSEP, le sépare en stems puis télécharge le résultat. Un compte et un jeton API sont nécessaires.

  1. Inscrivez-vous sur https://mvsep.com et copiez votre jeton API depuis votre profil.
  2. Ouvrez Paramètres > Outils externes > MVSEP et collez le jeton.
  3. Choisissez un modèle de séparation. Le Modèle vocal peut généralement rester sur sa valeur par défaut, et le Format audio définit le format des stems retournés.
  4. Dans l'assistant, cliquez sur Separate Vocals + Transcribe pour lancer le flux complet.
  5. Pour les chansons déjà ouvertes dans l'éditeur, utilisez Éditer > Séparer l'audio.

audio-separator (séparation vocale locale)

audio-separator exécute localement des modèles de séparation pré-entraînés sur votre machine, sans internet ni clé API. Il prend en charge les mêmes modèles MDX-Net, VR Architecture et Demucs qu'Ultimate Vocal Remover (UVR).

  1. Installez Python 3.9 ou plus récent.
  2. Installez audio-separator :
    • CPU uniquement : pip install audio-separator[cpu]
    • GPU NVIDIA : pip install audio-separator[gpu]
  3. Ouvrez Paramètres > Outils externes > audio-separator.
  4. Définissez l'exécutable Python de cet outil, ou laissez le champ vide pour utiliser le Python par défaut de Emplacements.
  5. Si vous utilisez Update package avec un champ Python spécifique vide, Yass crée automatiquement un environnement virtuel dédié à audio-separator et remplit le champ.
  6. Choisissez un modèle. Les modèles connus sont affichés avec un nom lisible ; les modèles inconnus gardent leur nom de fichier brut. Modèles recommandés :
    • Kim_Vocal_2.onnx - rapide, bonne qualité (par défaut)
    • MDX23C-8KFFT-InstVoc_HQ.ckpt - haute qualité
    • UVR-MDX-NET-Inst_HQ_3.onnx - alternative
    • BS-Roformer-Viperx-1297.ckpt - meilleure qualité, plus lent
  7. Vous pouvez définir un répertoire de cache des modèles. Les modèles téléchargés y sont stockés pour éviter de les retélécharger.
  8. Output format contrôle le format brut écrit par audio-separator avant que Yass ne convertisse les stems vers votre format audio cible.
  9. Cliquez sur Test configuration pour vérifier la configuration.
  10. Une fois la vérification réussie, le bouton Separate Vocals + Transcribe de l'assistant devient disponible. audio-separator est prioritaire sur MVSEP si les deux sont configurés.

OpenAI Whisper (transcription cloud)

L'API OpenAI Whisper transcrit l'audio avec des horodatages par mot et applique le résultat aux notes dans l'éditeur. Une clé API OpenAI est nécessaire.

  1. Obtenez une clé API sur https://platform.openai.com/api-keys.
  2. Ouvrez Paramètres > Outils externes > OpenAI et saisissez la clé.
  3. Choisissez un modèle de transcription (par ex. whisper-1) et, si besoin, un code de langue.
  4. Dans l'éditeur, utilisez Éditer > Align Notes (Transcription) pour transcrire et aligner la chanson actuelle.

WhisperX (transcription locale)

WhisperX exécute localement une transcription basée sur Whisper avec horodatage au niveau du mot. Pas d'internet ni de clé API nécessaires.

  1. Installez Python 3.9 ou plus récent.
  2. Installez WhisperX : pip install whisperx
  3. Ouvrez Paramètres > Outils externes > WhisperX.
  4. Définissez l'exécutable Python de WhisperX, ou laissez le champ vide pour utiliser le Python par défaut de Emplacements.
  5. Si vous utilisez Update package avec un champ Python WhisperX vide, Yass crée automatiquement un environnement virtuel dédié et enregistre cette runtime.
  6. Choisissez le mode d'invocation (python -m whisperx recommandé).
  7. Sélectionnez un modèle (par ex. large-v2) et votre périphérique (cpu ou cuda).
  8. Cliquez sur Test configuration pour lancer la vérification. Le résultat est enregistré et active le bouton Separate Vocals + Transcribe dans l'assistant.
  9. Dans l'éditeur, utilisez Éditer > Align Notes (Transcription) puis choisissez WhisperX.

Assistant de création : pipeline complet

L'assistant peut exécuter toute la chaîne automatiquement :

  1. Saisissez une URL YouTube sur la page YouTube et téléchargez l'audio.
  2. Sur la page Paroles, cliquez sur Separate Vocals + Transcribe.
  3. Yass convertit l'audio en WAV, sépare les voix (audio-separator ou MVSEP) et les transcrit (WhisperX).
  4. Les paroles sont remplies automatiquement. Vous pouvez les corriger avant de continuer.
  5. Terminez l'assistant pour créer la chanson avec des horodatages préremplis à partir de la transcription.

Prérequis : audio-separator (local, vérification réussie) ou MVSEP (jeton API configuré), plus WhisperX (vérification réussie).
