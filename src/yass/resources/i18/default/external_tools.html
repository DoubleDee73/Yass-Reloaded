External Tools

Yass Reloaded integrates with several optional external tools to enable vocal separation and lyrics transcription. Open Settings > External Tools to configure them.

FFmpeg

FFmpeg is required for audio format conversion (AAC, OGG, OPUS, etc.) and for the vocal separation workflow. Yass can auto-download FFmpeg, or you can point to an existing installation under External Tools > Locations.

  • Download: https://ffmpeg.org/download.html
  • On Windows, set the path to the bin folder containing ffmpeg.exe.

Python runtimes

Several local tools use Python. Under External Tools > Locations you can set a Default Python executable. Yass uses this as the shared fallback for lightweight Python helpers and as the bootstrap interpreter for creating tool-specific environments.

  • If a tool-specific Python field is left empty, Yass falls back to the default Python from Locations.
  • If you click Update package for WhisperX or audio-separator and no tool-specific Python runtime is set yet, Yass creates a dedicated virtual environment for that tool and stores the Python executable path there automatically.
  • This keeps heavyweight dependencies such as WhisperX and audio-separator isolated, while smaller Python helpers can still share the default Python.

yt-dlp

yt-dlp is used by the Create Song Wizard to download audio from YouTube and other platforms. No API key required.

  • Install: pip install yt-dlp or download the standalone binary.
  • Configure codec, resolution and audio format under Settings > Wizard > yt-dlp.

MVSEP (online vocal separation)

MVSEP is a cloud-based vocal separation service. It uploads your audio to the MVSEP API, separates it into stems and downloads the result. Requires an account and API token.

  1. Register at https://mvsep.com and copy your API token from your profile.
  2. Open Settings > External Tools > MVSEP and paste the token.
  3. Choose a separation model. The Vocal Model can usually stay on its default, and the Audio Format controls the returned stem format.
  4. In the Create Song Wizard, click Separate Vocals + Transcribe to run the full pipeline.
  5. For songs already open in the editor, use Edit > Separate Audio.

audio-separator (local vocal separation)

audio-separator runs pre-trained separation models locally on your machine — no internet or API key needed. It supports the same MDX-Net, VR Architecture and Demucs models used by Ultimate Vocal Remover (UVR).

  1. Install Python 3.9 or later.
  2. Install audio-separator:
    • CPU only: pip install audio-separator[cpu]
    • NVIDIA GPU: pip install audio-separator[gpu]
  3. Open Settings > External Tools > audio-separator.
  4. Set the Python executable for this tool, or leave it empty to use the default Python from External Tools > Locations.
  5. If you use Update package with an empty tool-specific Python field, Yass creates a dedicated audio-separator virtual environment and fills the field automatically.
  6. Choose a model. Yass shows known models with readable names; unknown ones fall back to the raw filename. Recommended models:
    • Kim_Vocal_2.onnx — fast, good quality (default)
    • MDX23C-8KFFT-InstVoc_HQ.ckpt — high quality
    • UVR-MDX-NET-Inst_HQ_3.onnx — alternative
    • BS-Roformer-Viperx-1297.ckpt — highest quality, slower
  7. Optionally set a model cache directory. Downloaded model files are stored there so they can be reused without downloading them again.
  8. Output format controls the raw stem format written by audio-separator before Yass converts the files to your configured target audio format.
  9. Click Test configuration to verify the setup.
  10. When health check passes, the Separate Vocals + Transcribe button in the Create Song Wizard becomes available. audio-separator is preferred over MVSEP when both are configured.

OpenAI Whisper (cloud transcription)

The OpenAI Whisper API transcribes audio to timestamped words and applies them to notes in the editor. Requires an OpenAI API key.

  1. Get an API key at https://platform.openai.com/api-keys.
  2. Open Settings > External Tools > OpenAI and enter the key.
  3. Choose a transcription model (e.g. whisper-1) and optionally a language code.
  4. In the editor, use Edit > Align Notes (Transcription) to transcribe and align the current song.

WhisperX (local transcription)

WhisperX runs Whisper-based transcription locally with word-level timestamps. No internet or API key required.

  1. Install Python 3.9 or later.
  2. Install WhisperX: pip install whisperx
  3. Open Settings > External Tools > WhisperX.
  4. Set the Python executable for WhisperX, or leave it empty to use the default Python from External Tools > Locations.
  5. If you use Update package with an empty WhisperX Python field, Yass creates a dedicated WhisperX virtual environment and stores that runtime automatically.
  6. Choose the invocation mode (python -m whisperx recommended).
  7. Select a model (e.g. large-v2) and your device (cpu or cuda).
  8. Click Test configuration to run the health check. The result is saved and used to enable the Separate Vocals + Transcribe button in the wizard.
  9. In the editor, use Edit > Align Notes (Transcription) and select WhisperX.

Create Song Wizard: full pipeline

The wizard can run the complete pipeline automatically:

  1. Enter a YouTube URL on the YouTube page and download the audio.
  2. On the Lyrics page, click Separate Vocals + Transcribe.
  3. Yass converts the audio to WAV, separates vocals (audio-separator or MVSEP) and transcribes them (WhisperX).
  4. The lyrics are filled in automatically. You can correct them before proceeding.
  5. Complete the wizard to create the song with timestamps pre-filled from the transcript.

Requirements: either audio-separator (local, health check passed) or MVSEP (API token configured), plus WhisperX (health check passed).
