Herramientas externas

Yass Reloaded puede integrarse con varias herramientas externas opcionales para separación vocal y transcripción de letras. Abre Configuración > Herramientas externas para configurarlas.

FFmpeg

FFmpeg es necesario para la conversión de audio (AAC, OGG, OPUS, etc.) y para el flujo de separación vocal. Yass puede descargar FFmpeg automáticamente, o puedes indicar una instalación existente en Herramientas externas > Ubicaciones.

  • Descarga: https://ffmpeg.org/download.html
  • En Windows, selecciona la carpeta bin que contiene ffmpeg.exe.

Entornos de Python

Varias herramientas locales usan Python. En Herramientas externas > Ubicaciones puedes definir un ejecutable Python predeterminado. Yass lo usa como valor compartido para ayudantes pequeños basados en Python y como intérprete base para crear entornos específicos por herramienta.

  • Si el campo Python de una herramienta está vacío, Yass usa el Python predeterminado de Ubicaciones.
  • Si pulsas Update package para WhisperX o audio-separator sin haber definido todavía un Python específico, Yass crea automáticamente un entorno virtual dedicado para esa herramienta y guarda allí la ruta del ejecutable.
  • Así, dependencias pesadas como WhisperX y audio-separator quedan aisladas, mientras que ayudantes más pequeños pueden compartir el Python predeterminado.

yt-dlp

yt-dlp se usa en el asistente de creación de canciones para descargar audio desde YouTube y otras plataformas. No requiere clave API.

  • Instalación: pip install yt-dlp o descarga el binario independiente.
  • Configura códec, resolución y formato de audio en Configuración > Asistente > yt-dlp.

MVSEP (separación vocal online)

MVSEP es un servicio en la nube para separación vocal. Sube tu audio a la API de MVSEP, lo separa en stems y descarga el resultado. Requiere una cuenta y un token API.

  1. Regístrate en https://mvsep.com y copia tu token API desde tu perfil.
  2. Abre Configuración > Herramientas externas > MVSEP y pega el token.
  3. Elige un modelo de separación. El Modelo vocal normalmente puede quedarse en el valor predeterminado, y el Formato de audio define el formato de los stems devueltos.
  4. En el asistente, pulsa Separate Vocals + Transcribe para ejecutar el flujo completo.
  5. Para canciones ya abiertas en el editor, usa Editar > Separar audio.

audio-separator (separación vocal local)

audio-separator ejecuta modelos de separación preentrenados localmente en tu equipo, sin internet ni clave API. Admite los mismos modelos MDX-Net, VR Architecture y Demucs usados por Ultimate Vocal Remover (UVR).

  1. Instala Python 3.9 o posterior.
  2. Instala audio-separator:
    • Solo CPU: pip install audio-separator[cpu]
    • GPU NVIDIA: pip install audio-separator[gpu]
  3. Abre Configuración > Herramientas externas > audio-separator.
  4. Indica el ejecutable Python para esta herramienta, o déjalo vacío para usar el Python predeterminado de Ubicaciones.
  5. Si usas Update package con el campo Python específico vacío, Yass crea un entorno virtual dedicado para audio-separator y lo rellena automáticamente.
  6. Elige un modelo. Los modelos conocidos se muestran con nombres legibles; los desconocidos usan el nombre del archivo. Modelos recomendados:
    • Kim_Vocal_2.onnx - rápido, buena calidad (predeterminado)
    • MDX23C-8KFFT-InstVoc_HQ.ckpt - alta calidad
    • UVR-MDX-NET-Inst_HQ_3.onnx - alternativa
    • BS-Roformer-Viperx-1297.ckpt - máxima calidad, más lento
  7. Opcionalmente, define un directorio de caché de modelos. Allí se guardan los modelos descargados para no tener que volver a descargarlos.
  8. Output format controla el formato bruto que escribe audio-separator antes de que Yass convierta los stems al formato de audio final configurado.
  9. Pulsa Test configuration para verificar la configuración.
  10. Cuando la comprobación sea correcta, el botón Separate Vocals + Transcribe del asistente quedará disponible. audio-separator tiene prioridad sobre MVSEP si ambos están configurados.

OpenAI Whisper (transcripción en la nube)

La API de OpenAI Whisper transcribe audio con marcas de tiempo por palabra y aplica el resultado a las notas del editor. Requiere una clave API de OpenAI.

  1. Obtén una clave API en https://platform.openai.com/api-keys.
  2. Abre Configuración > Herramientas externas > OpenAI e introduce la clave.
  3. Elige un modelo de transcripción (por ejemplo whisper-1) y, opcionalmente, un código de idioma.
  4. En el editor usa Editar > Align Notes (Transcription) para transcribir y alinear la canción actual.

WhisperX (transcripción local)

WhisperX ejecuta transcripción basada en Whisper localmente con marcas de tiempo por palabra. No requiere internet ni clave API.

  1. Instala Python 3.9 o posterior.
  2. Instala WhisperX: pip install whisperx
  3. Abre Configuración > Herramientas externas > WhisperX.
  4. Indica el ejecutable Python de WhisperX, o déjalo vacío para usar el Python predeterminado de Ubicaciones.
  5. Si usas Update package con el campo Python de WhisperX vacío, Yass crea un entorno virtual dedicado y guarda esa runtime automáticamente.
  6. Elige el modo de invocación (python -m whisperx recomendado).
  7. Selecciona un modelo (por ejemplo large-v2) y tu dispositivo (cpu o cuda).
  8. Pulsa Test configuration para ejecutar la comprobación. El resultado se guarda y habilita el botón Separate Vocals + Transcribe del asistente.
  9. En el editor, usa Editar > Align Notes (Transcription) y selecciona WhisperX.

Asistente de creación: flujo completo

El asistente puede ejecutar automáticamente toda la cadena:

  1. Introduce una URL de YouTube en la página YouTube y descarga el audio.
  2. En la página Letras, pulsa Separate Vocals + Transcribe.
  3. Yass convierte el audio a WAV, separa las voces (audio-separator o MVSEP) y las transcribe (WhisperX).
  4. La letra se rellena automáticamente. Puedes corregirla antes de continuar.
  5. Completa el asistente para crear la canción con marcas de tiempo precargadas desde la transcripción.

Requisitos: audio-separator (local, comprobación correcta) o MVSEP (token API configurado), y además WhisperX (comprobación correcta).
